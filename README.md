# 工业缺陷检测原型 (Industrial Defect Detection)

基于深度学习的工业表面缺陷检测原型系统，用于演示 AI 在工业质检场景的应用。

## 🎯 项目背景

工业视觉检测是 AI 落地的重要场景之一。传统方法依赖人工设计特征，而深度学习方法可以自动学习缺陷特征，适应更多样的检测场景。

本项目是一个**端到端的缺陷检测原型**，包含数据生成、模型训练、Web 演示全流程。

## 📦 项目结构

```
defect_detection/
├── data/               # 数据集目录
│   ├── good/          # 合格品图片
│   └── bad/           # 缺陷品图片
├── models/            # 训练好的模型权重
├── src/               # 源代码
│   ├── generate_data.py   # 数据生成脚本
│   ├── train.py           # 模型训练脚本
│   └── app.py             # Gradio 演示界面
├── requirements.txt   # Python 依赖
└── README.md          # 本文件
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 生成数据集

```bash
cd src
python generate_data.py ../data 100
```

生成 100 张合格品图片和 100 张缺陷品图片（包含划痕、斑点、脏污、裂纹四种缺陷类型）。

### 3. 训练模型

```bash
python train.py
```

训练约需 5-10 分钟（取决于硬件），训练完成后模型保存至 `models/` 目录。

### 4. 启动演示界面

```bash
python app.py
```

浏览器访问 `http://localhost:7860` 即可使用。

## 📊 技术细节

### 模型架构

- **骨干网络**：ResNet18（ImageNet 预训练）
- **分类头**：Dropout + FC(256) + ReLU + Dropout + FC(2)
- **输入尺寸**：224×224 RGB

### 数据增强

训练时使用以下增强提升泛化能力：
- 随机水平翻转
- 随机旋转（±10°）
- 颜色抖动（亮度、对比度）

### 训练配置

| 超参数 | 值 |
|--------|-----|
| Batch Size | 16 |
| 学习率 | 0.001 |
| 优化器 | Adam |
| 学习率调度 | ReduceLROnPlateau |
| Epochs | 20 |

## 📈 效果

使用模拟数据训练后，验证集准确率通常可达 **90%+**。

**注意**：这是演示原型，真实工业场景需要：
1. 采集真实产线图片
2. 专业标注（缺陷类型、位置）
3. 针对具体缺陷优化模型
4. 部署优化（TensorRT、ONNX 等）

2. **迁移学习**：使用预训练模型解决小样本问题
